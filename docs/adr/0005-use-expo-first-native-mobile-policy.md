# Use Expo-first native mobile policy

Offdex Mobile will use Expo and React Native by default, prefer Expo APIs for platform features, use Software Mansion libraries when they materially improve core UX or performance, and add custom native modules only where the platform boundary earns the cost. This keeps development fast without giving up native-quality interaction, performance, secure pairing, deep linking, background behavior, or input experiences where React Native alone is insufficient.
